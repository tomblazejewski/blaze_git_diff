A plugin to display git diffs for [Blaze Explorer](https://github.com/tomblazejewski/blaze-explorer)

# App Mappings

## Plugin level

| Default Mappings | Action              |
| ---------------- | ------------------- |
| `G`              | Open git diff popup |

## PopUp level

| Mappings | Action      |
| -------- | ----------- |
| `<Esc>`  | Close popup |
